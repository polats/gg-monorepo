# Product Overview

This is a monorepo containing two gaming applications and shared marketplace infrastructure:

## Applications

**Goblin Gardens** - A 3D physics-based gem collection and trading game with dual platform support (Reddit via Devvit and World via MiniKit). Players collect gems, grow their value over time, and trade with other players in a marketplace.

**Diamond Hands** - A World mini app featuring a raffle system with WorldID verification, wallet integration, and prize distribution. Built for the Worldcoin ecosystem.

## Shared Infrastructure

**Bazaar x402** - A reusable P2P marketplace library for virtual item trading using the x402 payment protocol on Solana. Provides framework-agnostic marketplace functionality with USDC payments.

## Key Concepts

- **Goblincore aesthetic**: Hoarding treasures, chaotic collection mechanics
- **Physics-based gameplay**: Real-time Rapier physics with drag-and-drop interactions
- **Player-driven economy**: Trading, marketplace, gem growth mechanics
- **Multi-platform**: Apps run on Reddit (Devvit), World (MiniKit), and local development
- **Cryptocurrency integration**: Solana/USDC payments, WorldID verification
